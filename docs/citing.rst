Citing bilby NR
===============

If you have used :code:`bilby_nr` as part of a project that leads to a
scientific publication, please add the following acknowledge statement:

.. code-block:: console

    Some of the results in this paper have been derived using the ``bilby_nr``
    package~\cite{Hoy:2022tst,Hoy:2024vpc}.

The citation in various formats is given below:

.. tabs::

    .. tab:: bibtex

        .. code-block:: bibtex

          @article{Hoy:2022tst,
              author = "Hoy, Charlie",
              title = "{Accelerating multimodel Bayesian inference, model selection, and systematic studies for gravitational wave astronomy}",
              eprint = "2208.00106",
              archivePrefix = "arXiv",
              primaryClass = "gr-qc",
              reportNumber = "LIGO-P2200228",
              doi = "10.1103/PhysRevD.106.083003",
              journal = "Phys. Rev. D",
              volume = "106",
              number = "8",
              pages = "083003",
              year = "2022"
          }

          @article{Hoy:2024vpc,
              author = "Hoy, Charlie and Akcay, Sarp and Mac Uilliam, Jake and Thompson, Jonathan E.",
              title = "{Incorporation of model accuracy in gravitational wave Bayesian inference}",
              eprint = "2409.19404",
              archivePrefix = "arXiv",
              primaryClass = "gr-qc",

              journal = "Nature Astron.",

          }
